09:30 | Palestra: NEWSQL: A Evolução do NoSQL para Processamento Escalável de Dados

Auditório do Centro de Ciências

Júlio Alcântara

Durante décadas, as bases de dados relacionais dominaram o mercado em relação ao armazenamento de dados para grandes sistemas. Como contrapartida, nos últimos anos, surgiram novos paradigmas de SGBD, cuja característica principal é oferecer algumas características complementares ao modelo relacional e às propriedades ACID. Um desses paradigmas é chamado NewSql, que atualmente representa a evolução do paradigma NoSql. A ideia principal do NewSql é apresentar características de desempenho escaláveis ​​do NoSql, mas também atuar e resolver uma das principais fraquezas encontradas no NoSql: preservar as características ACID de um sistema de gerenciamento de banco de dados convencional. Como principais tecnologias relacionadas com o NewSql, pode-se citar os bancos de dados "main-memory" e os novos protocolos de bloqueio e controle de concorrência. Esta tecnologia também permite o uso de linguagem SQL para lidar com o processamento de dados, o que é essencial para manter a compatibilidade com aplicativos existentes. Neste talk, falaremos sobre a evolução dos SGBDs nas últimas décadas e abrangeremos os principais paradigmas e desafios futuros.

09:30 | Palestra: Uma introdução (imperdível) à Visualização de Dados

Auditório do Centro de Ciências

Emanuele Santos

A computação, de modo geral, tem sido um grande acelerador para ciência, levando a uma explosão de informação nas mais diversas áreas. De acordo com algumas estimativas, em algum momento de 2006, a humanidade produziu somente naquele ano mais dados do que em todos os 40.000 anos precedentes. Entender e utilizar efetivamente os dados científicos atualmente produzidos constitui um dos grandes desafios de pesquisa do século XXI. Visualização é uma tecnologia chave nessa empreitada, pois ajuda as pessoas a explorarem e explicarem dados através de uma representação visual estática ou dinâmica. Uma suposição básica de visualização é que informação visual pode ser processada muito mais rapidamente do que números e texto (como diz o velho clichê: "Uma imagem vale mais que 1000 palavras").

Nesta palestra, você saberá o que é visualização de dados, como é o pipeline de visualização e alguns princípios e técnicas que já podem ser aplicados diretamente em apresentações, relatórios e artigos que envolvam análise de dados. Você também aprenderá quais as ferramentas comerciais e gratuitas mais utilizadas e verá dois exemplos que mostram o poder da Visualização de Dados em ação.

A computação forense visa obter informações através de análise de dados de um computador ou sistema, rede ou qualquer dispositivo de armazenamento de dados, que seja alvo de investigação, por crimes cibernéticos.

A computação forense é muito utilizada para desvendar crimes virtuais, roubos de identidade, crimes financeiros, ela age na busca de informações que não são vistas a olho nú, utilizando-se de técnicas especiais para rastrear informações.

Com o crescimento da tecnologia e uso de computadores, os problemas relacionados a segurança estão crescendo, então a computação forense está em pleno desenvolvimento em um mundo cada vez mais on-line.
